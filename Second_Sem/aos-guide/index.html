Fade анимации

Простые анимации появления с различными направлениями

<!-- Базовые fade анимации -->
<div data-aos="fade">fade</div>

<!-- Анимации с направлением -->
<div data-aos="fade-up">fade-up</div>
<div data-aos="fade-down">fade-down</div>
<div data-aos="fade-left">fade-left</div>
<div data-aos="fade-right">fade-right</div>
fade
Простое появление элемента с прозрачностью от 0 до 1.
fade-up
Элемент появляется, одновременно двигаясь вверх из начального положения на 20px.
fade-down
Элемент появляется, одновременно двигаясь вниз из начального положения на 20px.
fade-left
Элемент появляется, одновременно двигаясь влево из начального положения на 20px.
fade-right
Элемент появляется, одновременно двигаясь вправо из начального положения на 20px.

Flip анимации

Анимации переворота элементов

<!-- Flip анимации -->
<div data-aos="flip-up">flip-up</div>
<div data-aos="flip-down">flip-down</div>
<div data-aos="flip-left">flip-left</div>
<div data-aos="flip-right">flip-right</div>
flip-up
Элемент переворачивается по оси X, начиная с поворота на 180 градусов.
flip-down
Элемент переворачивается по оси X в обратном направлении.
flip-left
Элемент переворачивается по оси Y, начиная с поворота на 180 градусов.
flip-right
Элемент переворачивается по оси Y в обратном направлении.

Zoom анимации

Анимации масштабирования элементов

<!-- Zoom анимации -->
<div data-aos="zoom-in">zoom-in</div>
<div data-aos="zoom-in-up">zoom-in-up</div>
<div data-aos="zoom-in-down">zoom-in-down</div>

<div data-aos="zoom-in-right">zoom-in-right</div>

zoom-in
Элемент плавно увеличивается из точки с масштабом 0.5.
zoom-in-up
Элемент увеличивается, одновременно двигаясь вверх.
zoom-in-down
Элемент увеличивается, одновременно двигаясь вниз.
zoom-in-left
Элемент увеличивается, одновременно двигаясь влево.
zoom-in-right
Элемент увеличивается, одновременно двигаясь вправо.
zoom-out
Элемент плавно уменьшается до масштаба 0.5.

Кастомные анимации

Создавайте свои собственные анимации с помощью CSS

<!-- HTML для кастомной анимации -->
<div data-aos="custom-animation">custom-animation</div>
<div data-aos="rotate-animation">rotate-animation</div>
<div data-aos="custom-bounce">custom-bounce</div>
/* CSS для кастомной анимации */
[data-aos="custom-animation"] {
    opacity: 0;
    transform: scale(0.8);
    transition-property: transform, opacity;
}

[data-aos="custom-animation"].aos-animate {
    opacity: 1;
    transform: scale(1);
}

[data-aos="rotate-animation"] {
    opacity: 0;
    transform: rotate(-180deg) scale(0.5);
    transition: all 1s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

[data-aos="rotate-animation"].aos-animate {
    opacity: 1;
    transform: rotate(0) scale(1);
}

[data-aos="custom-bounce"] {
    opacity: 0;
    transform: scale(0.5);
    transition-property: transform, opacity;
}

[data-aos="custom-bounce"].aos-animate {
    opacity: 1;
    transform: scale(1.1);
    animation: bounce 0.5s ease infinite alternate;
}
custom-animation
Простая кастомная анимация с изменением масштаба и прозрачности.
rotate-animation
Анимация с вращением на 180 градусов и изменением масштаба.
custom-bounce
Анимация с эффектом "отскока" после появления.

Продвинутые техники

Комбинирование анимаций и сложные сценарии

<!-- Элементы с задержкой -->
<div data-aos="fade-up" data-aos-delay="0">0ms</div>
<div data-aos="fade-up" data-aos-delay="200">200ms</div>
<div data-aos="fade-up" data-aos-delay="400">400ms</div>
<div data-aos="fade-up" data-aos-delay="600">600ms</div>
<div data-aos="fade-up" data-aos-delay="800">800ms</div>

<!-- Анимация по якорю -->
<div id="anchor-element">Якорный элемент</div>
<div data-aos="fade-up" data-aos-anchor="#anchor-element">
    Анимируется при появлении якорного элемента
</div>
Элементы появляются последовательно с заданными задержками.
0ms
200ms
400ms
600ms
800ms
Элемент анимируется, когда якорный элемент появляется в зоне видимости.
Якорный элемент (появится первым)
Этот элемент анимируется после появления якорного элемента
